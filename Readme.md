# notes-node

a simple task manager write with nodejs

## Usage

adding a new task

```bash
$ node app.js add -t 'task name' -b 'task body'
```

removing a task

```bash
$ node app.js remove -t 'task name'
```

reading a task

```bash
$ node app.js read -t 'task name'
```

list all tasks

```bash
$ node app.js list
```
